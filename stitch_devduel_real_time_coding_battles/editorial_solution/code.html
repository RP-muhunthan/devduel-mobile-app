The Approach

The core challenge lies in determining the maximum visible skyline after applying N transformations. To achieve the O(N log N) threshold, we implement a Sweep-Line algorithm combined with a specialized segment tree.

We sort all transition events by their X-coordinates. As the sweep-line moves across the axis, we maintain an active set of heights in a Max-Priority Queue. For each change in the active set, we calculate the delta and append the coordinate if the global maximum has shifted.
