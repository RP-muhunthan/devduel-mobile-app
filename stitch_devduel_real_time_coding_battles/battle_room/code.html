Problem Statement

Given two sorted arrays nums1 and nums2 of size m and n respectively, return the median of the two sorted arrays.

The overall run time complexity should be O(log (m+n)).

Example 1

Input: nums1 = [1,3], nums2 = [2]

Output: 2.00000

Explanation: merged array = [1,2,3] and median is 2.

class Solution: def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float: # TODO: Implement O(log(m+n)) solution merged = sorted(nums1 + nums2) n = len(merged) if n % 2 == 0: return (merged[n // 2 - 1] + merged[n // 2]) / 2.0 else: return float(merged[n // 2])
